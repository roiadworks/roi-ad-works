# ROI Ad Works — Single Page Website

Files:
- index.html
- style.css
- script.js

## Publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload these three files to the repository root.
3. Open Settings → Pages.
4. Under Build and deployment, select "Deploy from a branch".
5. Select `main` and `/ (root)`, then Save.

## Lead form
The form currently uses FormSubmit and sends submissions to:
contactroiadworks@gmail.com

On the first submission, FormSubmit may send a confirmation email to activate the form endpoint. Confirm it before using the site for live ads.

## Tracking
Phone-click and form-submit hooks are included in script.js as placeholders. Add your Google tag / Google Tag Manager IDs before running paid traffic.

## Before launch
- Add your final domain/canonical URL when available.
- Add privacy policy / terms if required for your campaigns and jurisdiction.
- Replace or expand copy/case studies only with claims you can substantiate.
